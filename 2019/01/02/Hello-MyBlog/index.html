Hello MyBlog

你好啊，这是我的第一篇博客，今天来说说，为什么要写博客吧

学 IT ，最最重要的就是实践了，尤其是做项目过程中，很多坑坑洼洼等着我们去踩。
期间我们会学到很多技能，或者遇到一个 BUG ，经过折腾最后解决了，如果不及时进行总结，还是会遗忘的，而写博客就是一种对知识点总结的好方法，可以巩固加深印象。

写博客也是很锻炼表达能力的，怎样的博客才算好呢？简单地说就是把一个知识点用大白话讲解给一个菜鸟听，这就是好的博客，当然你也可以当做一种记录也行。

18 年初的时候，偶然认识一位 Java 大咖，他建议我写博客，因为面试的时候很加分。中途写了一半落下了，心思花在公众号上了，现在重新开始写，算是对自己的一种负责。

不管是写博客还是公众号，都是一种写作，摒弃急功近利的心态，坚持下去等积累到一定程度的时候会爆发出来的！
